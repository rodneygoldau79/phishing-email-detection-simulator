# Phishing Email Detection Simulator

This project simulates a security tool that analyzes email messages
for phishing indicators.

The program checks for common phishing techniques including:

- Suspicious links
- Urgent language
- Credential requests
- Unknown senders

Purpose:
Demonstrate phishing detection logic used in security monitoring
and email filtering systems.

## Phishing Indicators Detected

The simulation detects common phishing signals:

- Urgent language
- Credential requests
- Suspicious links
- Social engineering phrases

These techniques are frequently used in real phishing campaigns.
